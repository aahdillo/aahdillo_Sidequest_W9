Project Title

Side Quest Week 9: Fox Game Debug Mode

Group Members

Alexandra Dillon, aahdillo, 21092757

Description

This project lets users move a fox throughout a game by using keyboard inputs (left, right, and up arrow keys + space bar to attack).
The player can open the debug mode screen by pressing f and then toggle the gravity on or off by pressing m and the hitbox by pressing h

Interaction Instructions
Describe how to use or play your project.

As soon as the game starts the user can move around and toggle the gravity or hitbox modes on or off
